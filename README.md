# nebk
A toy loader and kernel for x86-64 architecture. **Just for fun.**

Welcome any issues.

## Development Environment
  * Windows 10
  * CMake 3.26.1
  * MSYS2 MinGW UCRT 64-bit toolchain, including:
    * gcc 13.2.0
    * nasm 2.16.01
    * mtools 4.0.43
    * qemu 8.2.0 
    * coreutils 8.32

Different version or runtime environment may work, but not tested.
